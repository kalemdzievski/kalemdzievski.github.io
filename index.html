4Stats 2015

Web page (HTML, CSS JavaScript, jQuery, AJAX, JSON, CanvasJS)

4Stats is a simple web application that uses Foursquare API to get data for Macedonian local places or as they are called by Foursquare API - Venues, and create simple statistics for the venues in different categories like: Top 5 venues by rating, users, tips and many others. Najdi Mesto 2014

Android application (Android, Java, SQLite, JSON, Foursquare)

Android application for finding local places for nightlife, coffee and eating in Macedonia, and showing the basic information about them, like: address, phone, price, rating, one random comment from a customer and location on map. The application "Najdi mesto" uses the API of FoursQuare to provide over 400 places for Skopje the capital city of Macedonia, and also many places for the other cities in Macedonia. The application also can show you what are the best places near you (radius of 1 km) and places that are popular in the moment.

Squares 2013

Puzzle Game (C#)

Squares is a simple game where the main goal is to make lines of 3 squares, and get get the highest. If you make a line of 3 squares you will get 100 points, or if you don’t make a line 5 new squares in 5 random colors will appear on the screen. So at the beginning there are 5 colors, and 5 squares are added if no you don’t make a line. Also there is timer, with countdown of 2 minutes. When the countdown is over new color is added and the number of squares added if you don’t make a line is increased by one.
